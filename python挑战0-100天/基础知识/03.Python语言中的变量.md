3.1 Python语言中的变量
变量是数据的载体，变量的值可以被读取和修改
数据类型包含：数值，文本、图像、音频、视频等各种各样的
不同类型的变量来表示数据类型的差异

3.1.1   整型（int)

print(0b100)  # 二进制整数
print(0o100)  # 八进制整数
print(100)    # 十进制整数
print(0x100)  # 十六进制整数

3.1.2   浮点型（float)

print(123.456)    # 数学写法
print(1.23456e2)  # 科学计数法

3.1.3   字符串型（str)
字符串是以单引号或双引号包裹起来的任意文本

3.1.4   布尔型（bool）
布尔型只有True、False两种值，可以用来表示现实世界中的“是”和“否”，命题的“真”和“假”，状况的“好”与“坏”，水平的“高”与“低”等等。变量的值只有两种状态，我们就可以使用布尔型
